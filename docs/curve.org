application.

The attractors here are all defined in the JSON file [[https://github.com/richmit/StrangeAttractorZoo/blob/main/src/zoo.json][zoo.json]].  The equations were
solved with *[[https://github.com/richmit/MRKISS][~MRKISS~]]*.
